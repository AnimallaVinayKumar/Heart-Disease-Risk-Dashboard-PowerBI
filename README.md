# Smart Healthcare Analytics – Heart Disease Risk Dashboard

## Project Overview
This project presents an interactive Power BI dashboard that analyzes heart disease risk using healthcare data.

## Features
- Total patient analysis
- Heart disease case identification
- Age group analysis
- Cholesterol and MaxHR visualization
- Gender-based risk analysis

## Tools Used
- Power BI
- Dataset: Heart Disease Dataset
- Data Visualization
